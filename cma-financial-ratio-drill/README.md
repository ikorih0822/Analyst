# 証券アナリスト 財務指標ドリル

財務分析で出題される主要な財務指標を、分子・分母の穴埋めで覚えるための静的Webアプリです。

## 使い方

- `index.html` をブラウザで開く
- 穴埋め画面で、分子と分母に入る項目を選んで `判定`
- 計算問題画面で、証券アナリスト試験を意識した数値問題を解く
- `復習` `まずまず` `定着` で自己評価を保存
- `弱点` 単元に、間違えた指標や復習指定した指標が集まる
- 教材編集画面で、指標の追加・修正・削除ができる
- JSON編集で、指標と計算問題をまとめて入れ替えられる
- Codexで問題を追加する場合は `codex-problems.js` に追記する

学習履歴はブラウザの `localStorage` に保存されます。
教材の編集内容も同じく `localStorage` に保存されます。

## 収録内容

- 収益性: ROE、ROA、売上高利益率など
- 安全性: 自己資本比率、流動比率、インタレスト・カバレッジ・レシオなど
- 効率性: 総資本回転率、棚卸資産回転率など
- 市場評価: EPS、BPS、PER、PBR、配当性向など
- 成長性、ROE分解式
- 計算問題: ROE、ROA、インタレスト・カバレッジ・レシオ、流動比率、PER/PBR、ROE分解、CCC、持続可能成長率、財務取引後の安全性指標など

## Codexで問題を増やす

`PROBLEM_AUTHORING.md` の形式に沿って、`codex-problems.js` に問題を追加します。
アプリ本体を大きく触らずに、指標を題材にした応用問題を増やせます。
